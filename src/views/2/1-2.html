    • 测试方法：滚动下面的列表，然后分别前往首页和详情页，然后再返回这里，可以注意观察滚动位置是否记住。

    • 详情页
    • 撑开高度，形成滚动
    • 详情页

    代码

    • 首页HTML：
      <a href="./1-2.html" rel="noreferrer">« 返回列表页</a>
      详情页HTML：
      <a href="./1-2.html">« 返回列表页</a>
    • 本页HTML：
      <div id="list" class="list">
    <ul>
        <li><a href="./detail.html">详情页</a></li>
        <li class="text">撑开高度，形成滚动</li>
        <li><a href="./detail.html">详情页</a></li>
    </ul>
</div>
    • JS代码：
      list.addEventListener('scroll', function() {
    localStorage.listScroll = this.scrollTop;
});

if (/detail\.html/.test(document.referrer)) {
    list.scrollTop = localStorage.listScroll;
} else {
    localStorage.listScroll = 0;
}
